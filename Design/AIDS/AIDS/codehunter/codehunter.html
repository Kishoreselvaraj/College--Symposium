Code Hunter

Technical Events

Code Hunter provides an enthralling gaming experience, allowing you to test and improve your coding proficiency across a diverse range of programming languages.

Eligibility:

All UG Engineering students can join.

Rules:

  • The event features questions presented in three primary programming languages: Python, C, and Java.
  • Participants are tasked with debugging provided code snippets in each of these languages.
  • Participants are required to provide appropriate answers based on their debugging analysis and corrections.
